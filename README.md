# CV-Boosted

Le framework [Orange Boosted](https://boosted.orange.com/), basé sur le très populaire [Twitter Boostrap](https://getbootstrap.com/), m'a permis de mettre en place rapidement mon [CV en ligne responsive](https://cv-abdelhak.netlify.app/). Les couleurs de la charte Orange ont été utilisées.

Ce site est déployé avec [Netlify](https://www.netlify.com/), qui détecte les modifications poussées vers ce [repository GitHub](https://github.com/AbdelhakBLD/CV-Boosted) et déclenche le déploiement automatisé.
